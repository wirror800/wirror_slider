
wirror_slider
========
首页轮播图插件  
* 多轮播引擎支持
* 后台管理便捷


使用方法：  
================
* 将插件拷贝到Discuz的插件目录
* 在后台应用管理中启用wirror_slider插件，并在“设置”中选择轮播引擎
* 在插件管理后台添加图片
* 在模板中插入diy变量
  

DISCUZ版本支持
================
* X2.5 X3.2
